# HieroTool
Hiero审核镜头工具

工具介绍：
    制作人员，如组长在审核组员镜头时，不用在cgteamwork中进行，查看镜头，可以直接通过该工具，将镜头文件与原素材
文件拖入到Hireo软件中进行对比查看。

    使用：
        1：直接在Hiero或NukeStudio激活该工具
        2：选择组号，显示出对应的原素材文件
        3：显示组员最新提交的dailies，可以一键同步版本
